## ⚡ Quick Start Guide

### Step 1: Read the Submission Guide (30 minutes)
📖 Open **WIKIPEDIA_SUBMISSION_GUIDE.md** and read it completely to understand Wikipedia's rules.

### Step 2: Gather Citations (1-2 weeks)
Collect these CRITICAL documents:
- ✅ NAAC Accreditation Certificate (from naac.gov.in)
- ✅ PCI Approval Letter (from pci.nic.in)
- ✅ University Affiliation Document (from tnmgrmu.ac.in)
- ✅ 2-3 news articles mentioning JKKN
- ✅ NIRF data (if available from nirfindia.org)

**This is THE MOST IMPORTANT STEP** - without proper citations, your article WILL be rejected.

### Step 3: Create Wikipedia Account (1 day)
- Go to en.wikipedia.org
- Create account with professional username
- Make 4-5 minor edits to other articles (builds trust)
- Declare conflict of interest on your user page

### Step 4: Draft in Sandbox (2-3 days)
- Copy content from **WIKIPEDIA_ARTICLE_DRAFT.md**
- Add all citations using Wikipedia citation format
- Preview to ensure formatting is correct

### Step 5: Submit for Review (1 day)
- Use Articles for Creation (AfC) process
- Wait 2-6 weeks for volunteer reviewer
- Monitor your user talk page for feedback

### Step 6: Create Wikidata (2 hours)
- Once Wikipedia article is approved
- Use **WIKIDATA_ENTITY_STRUCTURE.md** as template
- Create entity on wikidata.org
- Link to Wikipedia article

## 🚨 Critical Success Factors

### Must-Haves for Approval:
1. ✅ **3+ Independent Sources** (news articles, government documents)
2. ✅ **Neutral Tone** (no promotional language like "best", "premier")
3. ✅ **Inline Citations** (every fact must be cited)
4. ✅ **Encyclopedic Style** (write like an encyclopedia, not a brochure)
5. ✅ **Conflict of Interest Disclosure** (be transparent about affiliation)

### Common Rejection Reasons:
❌ Promotional tone
❌ Insufficient independent sources
❌ No inline citations
❌ Using only college website as source
❌ Reads like advertising
